# Actividad API Conductores y Automóviles

## Requisitos

* Node.js
* PostgreSQL
* pgAdmin

## Instalación

1. Crear base de datos llamada `actividad2`.
2. Ejecutar el archivo `actividad2.sql`.
3. Entrar a la carpeta backend.

```bash
npm install
```

4. Configurar el archivo `.env`.

5. Ejecutar:

```bash
node index.js
```

## Uso

Abrir el archivo:

```text
frontend/index.html
```

y utilizar los botones para consumir la API.
